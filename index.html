Jogo do Pong

Esse jogo foi desenvolvido a partir do Curso ofertado pelo Alura: Lógica de programação: desenvolvendo um jogo estilo Pong

O curso ensina a fazer o jogo do Pong,como forma de desenvolver e aprimorar a lógica de programação,mas ele não ensina a construir o site da maneira que ele está agora (ele não ensina a construção do site como está,somente a integrar JS,que eu utilizei para melhorar o projeto desenvolvido no curso).Espero que goste do jogo!se houver algum feedback me contate pelo Git Hub JPMOURA7

Controles do Jogo:

Para controlar a raquete à esquerda: aperte seta para cima para move-la para cima e seta para baixo para move-la para baixo.

Para controlar a raquete à direita: aperte W para move-la para cima e S para move-la para baixo.
